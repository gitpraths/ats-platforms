Placement Details

One-click placement creation with automated welfare check tracking

✓ Completed
  • ✓
    Employment Confirmation Email to Employer When a candidate is placed, an automated confirmation email is sent to the employer immediately. → Job Detail › Applications tab › Place button
  • ✓
    One-Click Placement + Job Start Date Any application can be converted to a placement in one click. A start date and optional notes are entered at the time of placement. → Job Detail › Applications tab › Place
  • ✓
    Tracking from Job Start Date Welfare checks are automatically scheduled from the placement start date: Day 1, Week 1, Month 1, Month 3, and Month 6. → Sidebar › Placements › click any placement
  • ✓
    Indicator / Email / Notifications at Required Timeline Colour-coded dot indicators show welfare check status (due / completed / overdue). Automated emails are sent to the employer at each milestone. → Placements list › welfare check dots
  • ✓
    Job Welfare Checks — Automated Emails to Employer At each welfare check milestone, an automated email is sent to the employer requesting confirmation of the candidate's ongoing employment. → Placement Detail › mark welfare check complete
✅ All placement and welfare check requirements are fully implemented and live.

Reports

Date-filtered analytics with CSV export across three report types

✓ Completed
  • ✓
    Provider Wise Report Shows all providers with total candidates, active, placed, job seeking, inactive counts, and placement rate. Filterable by date range. CSV export available. → Sidebar › Reports › Provider Report tab
  • ✓
    Placement Tracking Report Lists all placements with candidate name, job, employer, provider, start date, employer confirmation status, and welfare check indicators. Filterable by date range, employer, and provider. CSV export available. → Sidebar › Reports › Placement Tracking tab
  • ✓
    Staff Wise Report Shows each staff member's assigned jobs, active jobs, total applications managed, and total placements made. Filterable by date range. CSV export available. → Sidebar › Reports › Staff Report tab
  • ✓
    Search / Filter Button Date range and filter dropdowns only trigger a data refresh when the Search button is clicked — preventing accidental partial queries. → Reports › Search button
  • ✓
    CSV Export All three report tabs support one-click CSV download. Files open correctly in Excel and Numbers. → Reports › Export CSV button (top right)
✅ All reporting requirements are fully implemented and live.
