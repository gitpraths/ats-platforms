## Recording Tips

- Use **1280x720** or **1920x1080** window size for clean recording
- Zoom browser to **110%** so text is readable
- Use **Loom** or **QuickTime** (Cmd+Shift+5) to record
- Log in as **admin@myats.dev** unless the script says otherwise
- Keep mouse movements **slow and deliberate**
- Pause 1-2 seconds after each click before narrating
