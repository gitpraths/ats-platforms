Vacancy Form — What We Are Building

What Is Changing

The current “Create Job” form is being redesigned into a new “Add Vacancy” form.

  • Current form: 4 steps
  • New form: 3 steps

Vacancies Tab

The Vacancies page will show two buttons at the top:

  • + Add Vacancy
  • + Add Employer

Step 1 — Vacancy Details

Field Notes
Vacancy ID Auto-generated. Read-only. Shown at top of form
Job Title Required
Employer Dropdown from Employer list. Shows “+ Add Employer” link if not found
Industry Text field
Pay Rate $ per hour
No. of Positions Number
Work Type Dropdown: Full-time / Part-time / Casual
Work Location Text field
Job Board URL Pre-filled with WorkVision URL

Step 2 — Description & Compliance

Field Notes
Job Description Large text area
Police Check Yes / No / Not Required
Drug & Alcohol Test Yes / No — marked as Optional
WWC Yes / No
Car Required Yes / No
Public Transport Accessible Yes / No
Wage Subsidy Yes / No
Comments Text area

Step 3 — Candidate Assignment

Field Notes
Candidate Search from existing candidates. Shows “+ Add Candidate” link if not found
Interview Date Date picker
ETS Date Date picker
Placement Date Date picker

Fields Being Removed

These fields exist in the current form and will be removed:

  • Department
  • Work Model (Onsite / Remote / Hybrid)
  • Team
  • Type of Vacancy
  • Staff Working Status
  • End Date
  • Required Skills
  • Desired Skills
  • Min Annual Salary
  • Max Annual Salary
  • Currency
  • Minimum Experience (years)
  • Application Deadline
  • Cover Letter Required
  • Recruiter Assignment

One Rule

When a vacancy status is set to Archived, the Job Board URL is automatically cleared.
